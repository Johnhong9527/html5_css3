常用数组方法

concat(< otherArray >)

将数组和参数所指数组的内容合并为一个新数组.可指定多个数组 => 数组

join(< separator >)

将所有数组元素连为一个字符串.各元素内容用参数指定的字符分割 => 字符串

pop()

把数组当做栈使用,删除并返回数组的最后一个元素 => 对象

push(< item >)

把数组当做栈使用,将指定的数据添加到数组中 => void

reverse()

就地反转数组元素的次序 => 数组

shift()

累死 pop,但操作的是数组的第一个元素 => 对象

slice(< start >,< end >)

返回一个子数组 => 数组

sort()

就地对数组元素排序 => 数组

unshift(< item >)

类似push,但新元素被插到数组的开头位置

处理错误

try{}catch(err){}语句处理错误

<script>
  var txt = "";
  function message() {
    try {
      adddlert("Welcome guest!");
    }
    // catch提供一个从错误中恢复或在错误发生之后进行清理工作的机会
    catch (err) {
      /*
      error:
        message: 对错误条件的说明 => 字符串
        name: 错误的名称,默认为error => 字符串
        number: 该错误的错误代码(如果有的话) => 数值
      */
      txt = "本页有一个错误。\n\n";
      txt += "错误描述：" + err.message + "\n\n";
      txt += "点击确定继续。\n\n";
      alert(txt);
    }
  }
</script>
<input type="button" value="查看消息" onclick="message()"/>

finally子句

接上:如果错误语句,有无报错都需执行语句,使用finally子句即可 
  <script>
  var txt = "";
  function message() {
    try {
      // 时间事件响应代码
    }
    catch (err) {
      // 时间事件响应代码
    } finally{
      // 时间事件响应代码
    }
  }
</script>
<input type="button" value="查看消息" onclick="message()"/>
